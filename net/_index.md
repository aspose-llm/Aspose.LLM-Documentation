## Welcome to Aspose.LLM for .NET

![Aspose.LLM for .NET](aspose-llm-net.png)

**Aspose.LLM for .NET** lets you integrate large language models into your .NET applications. Create an API instance from a preset (e.g. Qwen 2.5, Gemma, Llama, Phi), start chat sessions, send messages, and optionally save or load conversation state. You can use default presets or customize parameters for your scenario.

The library is implemented in C# and supports running models on CPU or GPU. Get started with [system requirements](/net/system-requirements/), [installation](/net/installation/), and a [simple example](/net/hello-world/).
